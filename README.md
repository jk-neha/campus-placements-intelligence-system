# 🌍 Overview

## Campus Placement Intelligence System

Campus Placement Intelligence System is a full-stack web application designed to simplify and automate the placement process inside educational institutions.

The platform helps students maintain placement profiles, track applications, manage company opportunities, and enables placement coordinators to efficiently monitor recruitment activities.

The system provides a centralized solution replacing traditional spreadsheet-based placement management.

---

# ✨ Features

| Feature | Description |
|:---|:---|
| 🔐 Authentication | Secure student and admin login system |
| 👨‍🎓 Student Dashboard | Manage profile, skills, academics and placement status |
| 🏢 Company Management | Add and manage company placement drives |
| 📄 Resume Management | Upload and maintain student resumes |
| 📊 Placement Tracking | Track applications, selections and recruitment progress |
| 🎯 Eligibility Checking | Identify eligible students for company requirements |
| 📈 Dashboard Analytics | Visual insights into placement activities |
| 🔄 CRUD Operations | Complete create, read, update and delete functionality |
| 🌐 Full Stack Architecture | Modern frontend + backend separation |
